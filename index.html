Projects

Responsive website

The main focus of this project was creating a responsive website about a given subject - in this case, flat design. It is the second website I have ever created and my first attempt at achieving responsiveness. It is not included here because I feel particularly proud about this project - but because it's now been one year since I created it and it marks the beginning of my journey with coding, as well as helps to highlight the progress I have made during this time.

Onboarding game

This onboarding project is targeted at new and future students of Copenhagen School of Design and Technology. Using gamification techniques is meant to make learning things about university easier and more pleasant, as well as rewarding. Taking a quiz during each "level" helps the user to memorise newly acquired informations and serves as additional motivation. The final solution is a result of teamwork.

Danish movies

A simple and classical website dedicated to various Danish movies. The purpose of the project was mainly working with dynamic data and fetching it, as well as filtering it by given properties. It is supposed to serve as a little guide for movie enthusiasts and newcomers to Denmark, who are interested in broadening their knowledge about Danish culture.

Kopenhagen remake

Redesign made for a client - kopenhagen.dk. The goal was to achieve a minimalistic yet modern look that embodies art. Most of the data is dynamic, with Wordpress used as REST API. The other main component of this project is a lot of JavaScript - both to display the data, hide it and filter it.

Hogwarts student list

The latest and probably most complex project in regards to working with JavaScript - it revolves around Harry Potter universe and lets the user perform various actions - such as expelling the students, adding them to inquisitorial squad or giving them a prefect status. PS. did you ever wonder what happens when you try to hack Hogwarts? Wonder no more, just write "hackTheSystem()" in the console ;)
